asdf

管理多个运行环境版本的简单命令行工具

快速上手 项目简介

一个工具

使用单个命令行工具和命令界面管理你的每个项目运行环境。

插件

现有运行环境和工具的大型生态系统。简单 API 用于根据需要添加对新工具的支持！

向后兼容

支持从现有配置文件 .nvmrc、.node-versions、.ruby-version 平滑迁移！

一个配置文件

一个可共享的 .tool-versions 配置文件管理所有工具、运行环境及其版本。

Shells

支持 Bash、ZSH、Fish 和 Elvish，并提供补全功能。

GitHub Actions

提供 Github Action 在 CI/CD 工作流中安装和使用 .tool-versions。
